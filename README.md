# WinScript

WinScript is a an open-source tool that allows you to build a Windows script from scratch. It features security, privacy, and performance tweaks, along with the ability to install applications through the terminal.

<img src="animation.gif">

## Features

- 🧹 **Debloat**: Remove unnecessary bloatware to free up system resources.
- 🔏 **Privacy**: Protect your privacy by disabling telemetry & other settings.
- 🚀 **Performance**: Optimize your system for better performance.
- ⬇️ **Install Apps**: Easily install 3rd-party applications through chocolatey.
